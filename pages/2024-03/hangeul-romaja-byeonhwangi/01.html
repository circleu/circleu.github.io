한글 로마자 변환기

2. 만들기

최근 수정: 2024-03-16 21:30

계획

만들기 전에 프로그램을 어떤 식으로 짤 지 구상해 봅시다.

대충 이런 절차로 만들어 보겠습니다:

  1. 한글을 입력 받고 utf-8로 변환한다.
  2. utf-8을 유니코드로 변환한다.
  3. 한글은 유니코드 테이블에서 일정한 규칙으로 배열되어 있기에 반복문을 돌려서 한글을 분해한다.
  4. 분해한 한글을 로마자로 변환한다.
  5. 로마자 표기법에 따라 바꿔야 할 부분들을 수정한다.

제작

입력 받기

먼저 한글을 입력 받아 봅시다.

input 함수를 사용하여 문자를 입력 받습니다. 테스트용 문장은 "안녕하세요."로 해보겠습니다.

text = input()
print(text)
>>> 안녕하세요.

이런 식으로 출력이 됩니다.

utf-8로 변환하기

encode 메소드를 사용하면 문자를 인코딩할 수 있습니다.

encodedText = text.encode(encoding="utf-8")
print(encodedText)
>>> b'\xec\x95\x88\xeb\x85\x95\xed\x95\x98\xec\x84\xb8\xec\x9a\x94.'

이렇게 출력이 됩니다.

그리고 hex 메소드를 사용하여 바이트 문자열을 문자열로 변환하고 split 메소드를 사용하여 리스트로 만듭니다.

encodedText = encodedText.hex(" ").split(" ")
print(encodedText)
>>> ['ec', '95', '88', 'eb', '85', '95', 'ed', '95', '98', 'ec', '84', 'b8', 'ec', '9a', '94', '2e']
