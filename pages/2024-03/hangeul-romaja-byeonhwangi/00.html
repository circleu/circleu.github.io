페이지 이름 변환하는 데에 쓰기 위해서, 그리고 개인적인 흥미(한국어와 한글에 관심이 많습니다.)를 이유로 만들게 되었습니다.

한글과 유니코드의 표현

우선 C언어에서 한글이 어떤 식으로 표현되는지 알아보겠습니다.

한글은 utf-8로 인코딩하였을 때 3 바이트입니다. 그래서 배열에 한글을 저장할 때에는 배열의 크기를 3*n+1 바이트(n은 글자 수, +1은 널문자)로 선언하면 됩니다. locale.h 헤더에 선언된 wchar_t 자료형을 사용해도 되지만 번거로운 부분이 많기에 이 글에서는 사용하지 않겠습니다.

#include <stdio.h>


int main(void) {
 char hello[16] = "안녕하세요";
 printf("%s", hello);

 return 0;
}
